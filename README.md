## Usage

```bash
md-color --help

Usage: md-color [OPTIONS]

Options:
  -f, --file <FILE>      get color from image file [default: ]
  -o, --output <OUTPUT>  result output file, default is print to stdout [default: ]
  -s, --source <SOURCE>  get color from source color [default: ]
      --dark             is dark mode
  -h, --help             Print help
  -V, --version          Print version
```
